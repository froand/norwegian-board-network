# Norwegian Board & Government Network Explorer

Interactive graph visualization showing relationships between Norwegian board members, politicians, and government officials.

## Data Sources
- **Brønnøysundregistrene (brreg.no)** — Company roles and board members
- **Stortinget.no** — Politicians and committees
- **Regjeringen.no** — Government positions

## Tech Stack
- Frontend: React + TypeScript + React Force Graph
- Backend: Express + TypeScript
- Styling: Tailwind CSS

## Getting Started

```bash
npm run install:all
npm run dev
```

Frontend runs on http://localhost:5173
Backend runs on http://localhost:3001

See `infra/` folder for Azure deployment configuration.
